Recent Work

A collection of projects I've worked on.

Smart Water Monitoring for Healthier Fish and Smarter Aquaculture.

Aqua Mate

Aqua Mate is an IoT-powered monitoring system designed to help fishermen and hatchery owners manage fish ponds with precision. Using an ESP32 microcontroller and sensors for pH, temperature, and dissolved oxygen, it continuously collects water quality data and streams it to a cloud-based dashboard. The system provides real-time alerts, historical insights, and email notifications, enabling proactive decisions that improve fish health and productivity. With features like multi-hatchery management, secure device linking, and data visualization, Aqua Mate delivers a low-cost, scalable solution for modern aquaculture in Bangladesh.

Blood Bucket. Connecting Lives Through Technology

Blood Bucket

Blood Bucket is a web-based blood donation management platform designed to quickly connect patients in need with eligible blood donors. The system allows users to request blood, register as donors, and receive automated email notifications based on blood group compatibility and donation eligibility. With features like an admin dashboard, donor management, request tracking, and a public blog system for awareness, Blood Bucket focuses on speed, reliability, and community impact. The project demonstrates real-world problem solving using PHP, MySQL, and modern web technologies to support lifesaving decisions in critical moments.

Planting Trees Digitally to Grow a Greener Tomorrow.

Green Earth

I developed a fully responsive Plant a Tree Campaign website using HTML, CSS (Tailwind & DaisyUI), and Vanilla JavaScript. The project integrates dynamic API endpoints to load categories, display plants in a 3-column card layout, and show detailed information in modals. Key features include an interactive cart system with total price calculation, category-based filtering, and a loading spinner for smooth user experience. The site also highlights campaign impact through statistics and provides a form for users to pledge trees, making it both functional and engaging.

Instant Access to Emergency Help, When Every Second Matters

Emergency Hotline

This project is a mobile-first web dashboard designed to provide fast and reliable access to essential emergency service contact numbers in Bangladesh. The main goal is to help users take quick action during critical situations by enabling one-tap calling and instant number copying from any mobile device. The clean, responsive interface ensures smooth usability on smartphones, where emergencies are most likely to occur. Additional features such as call history and interaction tracking improve user awareness and experience. Overall, the dashboard focuses on accessibility, speed, and public safety through a modern mobile web solution.

A Secure, Simple, and Smart Digital Wallet Experience for the Mobile Web.

Payooo!

This project is a mobile-style digital wallet web application built using HTML, Tailwind CSS, DaisyUI, and Vanilla JavaScript. It includes a secure login system using a mobile number and PIN, followed by core wallet features such as adding money, cash withdrawal, balance management, and logout functionality. The application validates user inputs to prevent incorrect transactions and dynamically updates the available balance in real time. A transaction history module records and displays recent activities with timestamps for better tracking. Overall, the project demonstrates practical front-end logic implementation, state handling, and user-focused financial workflow design suitable for a mobile web experience.
